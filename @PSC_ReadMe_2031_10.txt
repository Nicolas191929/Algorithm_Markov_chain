Title: Simple Sample of a Markov Chain Algorithm
Description: A markov chain is a sequence of random variables in which the future variable is determined by the present variable but is independent of the way in which the present state arose from its predecessors. The code is a simple example of a markov chain that generates somewhat random text output from a given text input.
